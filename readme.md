Japanese Utils
==============

概要
----

Japanese Utils
は、日本語特有の処理に関するユーティリティ・クラスを提供します。特にさまざまな文字を使い分ける日本語における文字処理は時として煩雑であり、要件が単純な割りに実装に手間取ることも少なくありません。Japanese
Utils はそういった局面で活用されることを目標としています。
